# Google GSON Example

An example of using Google JSON library GSON to read/write JSON structures from/to the local file system.

## Tools Used

| Tool     |  Version |
|:---------|---------:|
| Java     |   23.0.2 |
| IntelliJ | 2024.3.4 |
| VSCode   |   1.97.2 |
| GSON     |   2.12.1 |
